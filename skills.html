Software and Tools:
 I am well versed with a number of data repositories and the corresponding APIs, and other data sources (e.g. GitHub, GHTorrent, World of Code, Google Big Query etc.). I have experience in using both relational (MySQL) and non-relational (NoSQL, e.g. MongoDB) databases for data storage. I also am familiar with useful utilities like docker, jupyter notebooks, vim, R-Studio, R-Shiny etc. and the git version control system. In terms of documentation, I am well-acquainted with LaTeX and Markdown styles. Machine Learning Techniques:
In terms of the supervised machine learning models, I am knowledgeable about Linear and Logistic Regression, Decision Trees and Random Forest, GAM (Generalized Additive Model), SVM (Support Vector Machine) etc. Among various unsupervised modeling techniques, I am proficient in different clustering methods (e.g. K-means, Fuzzy C-means, PAM, Hierarchical clustering etc.), dimension reduction techniques like PCA and SVD, association rule mining, and different discretization techniques. I am well familiar with Bayesian Network (BN) models, especially BN structure search techniques and inferencing (both causal and non-causal), and I also have some experience in using deep learning models, like the MLP (Multi Layer Perceptron) model, though mostly on numerical data. I primarily use R for modeling, but I am familiar with similar functions in Python as well.
